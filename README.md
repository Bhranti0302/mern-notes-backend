# MERN Notes App

A full-stack Notes Application built using the MERN stack.
This project is being developed to practice and understand full-stack development concepts including authentication, REST APIs, and database integration.

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication
* Mongoose

## Features (In Progress)

* User authentication (Signup / Login)
* Secure password hashing using bcrypt
* JWT-based authentication
* Create, read, update, delete notes
* Protected routes
* Pagination and search for notes
* Responsive UI

## Project Structure

mern-notes-backend
mern-notes-frontend

## Installation

### Backend

1. Clone the repository
2. Install dependencies

npm install

3. Create a `.env` file
4. Start the server

npm run dev

### Frontend

1. Install dependencies

npm install

2. Start React app

npm start
